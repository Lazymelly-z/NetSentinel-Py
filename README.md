# NetSentinel-Py
## Uses Python and Scapy as a lightweight intrusion detection system

## Features :
* **Auto-Interface Selection:** Automatically detects the active network card being used for internet connection.
* **ARP Spoofing Detection  :** Monitors ARP tables to detect "Man-In-Middle" attacks
* **Trapports               :** Detects unauthorized connection attempts on ports commonly used in attacks
* **Live Debugging          :** Gives a real-time summary for network analysis

## Requirements :
* **Python 3.x**
* **Npcap**
* **Scappy Libary**

## Installation :
1. Install Npcap (https://nmap.org/npcap/).
2. Install the required Python library : Scapy
